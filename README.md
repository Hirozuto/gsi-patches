## xdroidOSS Twelve Patches for Generic System Image
Patches for building xdroidOSS Twelve

### How to apply patches? ###
```bash
git clone https://github.com/Hirozuto/xdroid-patches -b thirteen
bash xdroid-patches/apply.sh
```
